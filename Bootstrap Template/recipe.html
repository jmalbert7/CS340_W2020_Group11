Review Your Order

Please verify your address and payment information before placing your order.

Address Information

Values in this form will be populated with data extracted from JSON string received from backend server via SELECT command on Addresses table.
When button is clicked, popup form will appear where user can modify the address field(s). On click, send UPDATE command to Addresses table WHERE customer_id = this.customer_id.


Payment Information

Values in this form will be populated with data extracted from JSON string received from backend server via a SELECT command on Payment_Informations table.
When button is clicked, popup form will appear where user can modify the payment field(s). On click, send UPDATE command to Payment_Informations table WHERE customer_id = this.customer_id.
